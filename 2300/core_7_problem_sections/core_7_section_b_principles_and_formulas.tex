    \item
    Select all the statements which may be proven using the pigeonhole principle:
    \begin{enumerate}
        \item If there are 12 chores at home and 4 family members, at least one family member must do two chores.
        \item A function from a set of size 10 to a set of size 7 must not be injective.
        \item A hashing function that maps 40 keys to 20 hashes must have at least one collision.
        \item Dogs like donuts.
    \end{enumerate}
    \item
    Select all the statements which may be proven using the generalized pigeonhole principle:
    \begin{enumerate}
        \item If there are 12 chores at home and 4 family members, at least one family member must do 4 chores.
        \item If there are 12 chores at home and 4 family members, at least one family member must do 3 chores.
        \item For every (nontrivial) function \( f : A \to B \), there exists a \( b \in B \) such that \( |f^{-1}(b)| \geq \frac{|A|}{|B|} \).
        \item Dogs like 3 donuts.
        \item A hashing function with \( n \) keys and \( d \) hashes has a hash with at least \( \frac{n}{d} - 1 \) collisions.
        \item A hashing function with \( n \) keys and \( n \cdot d \) hashes has a hash with at least \( \frac{1}{d} - 1 \) collisions.
    \end{enumerate}
    \item
    \textbf{Question:} How many permutations are there of the letters \( a, b, c, d, e, f, g, h, i \)?
    Select \textbf{all} that apply:
    \begin{enumerate}
        \item \( 9! \)
        \item 9 times the number of ways we may permute 8 letters
        \item 42
        \item \( 7! \)
    \end{enumerate}
    \item
    In which scenarios may you want to use \( P(n, k) \)?
    \begin{enumerate}
        \item You would like to find how many ways you can pick the president, vice president, and secretary from a club of 10 members.
        \item You would like to know how many different road trips there are which bring you to 3 out of 6 neighboring cities.
        \item You would like to count the number of injective functions \( f : \{1, 2, 3\} \to \{1, 2, 3, 4, 5, 6, 7\} \).
        \item You would like to count the number of surjective functions \( f : \{1, 2, 3\} \to \{1, 2, 3, 4, 5, 6, 7\} \).
        \item You would like to count the number of dogs that eat donuts.
    \end{enumerate}
    \item
    Select all of the statements below which are true:
    \begin{enumerate}
        \item $\displaystyle \binom{19}{8}=\binom{18}{7}+\binom{18}{8}$
        \item $\displaystyle \binom{20}{8}=\binom{19}{7}+\binom{19}{8}$
        \item $\displaystyle \binom{8}{8}=1$
        \item $\displaystyle \binom{\text{dogs}}{\text{donuts}}=\binom{\text{this}}{\text{is}}+\binom{\text{one}}{\text{wrong}}$
        \item $\displaystyle \binom{20}{8}=\binom{19}{7}+\binom{18}{8}+\binom{18}{7}$
        \item $\displaystyle \binom{19}{1}=19$
    \end{enumerate}
